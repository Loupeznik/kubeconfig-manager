## kcm context rename

Rename a context and move its per-context state (tags, alerts) to the new name

```
kcm context rename <file> <old-name> <new-name> [flags]
```

### Options

```
      --dir string   Kubeconfig directory (default: ~/.kube)
  -h, --help         help for rename
```
