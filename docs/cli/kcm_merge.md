## kcm merge

Merge two kubeconfig files into a new file

```
kcm merge <a> <b> <out> [flags]
```

### Options

```
      --force                Overwrite the destination if it exists
  -h, --help                 help for merge
      --on-conflict string   error | skip | overwrite (default "error")
```
