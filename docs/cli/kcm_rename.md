## kcm rename

Rename a kubeconfig file on disk (metadata re-binds automatically)

```
kcm rename <file> <new-name> [flags]
```

### Options

```
      --dir string   Kubeconfig directory (default: ~/.kube)
      --force        Overwrite destination if it exists
  -h, --help         help for rename
```
