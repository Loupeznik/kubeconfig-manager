## kcm show

Show contexts, clusters, and users of a kubeconfig file

```
kcm show <name-or-file> [flags]
```

### Options

```
      --dir string   Kubeconfig directory for bare-name lookups (default: ~/.kube)
  -h, --help         help for show
```
