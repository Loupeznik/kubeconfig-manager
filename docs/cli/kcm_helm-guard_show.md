## kcm helm-guard show

Show the effective helm guard policy (global, or --file-specific)

```
kcm helm-guard show [flags]
```

### Options

```
      --dir string    Kubeconfig directory (default: ~/.kube)
      --file string   Show policy for this kubeconfig (default: global only)
  -h, --help          help for show
```
