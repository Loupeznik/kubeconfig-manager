## kcm use

Print a shell snippet that exports KUBECONFIG to the selected file

```
kcm use <name-or-file> [flags]
```

### Options

```
      --dir string     Kubeconfig directory (default: ~/.kube)
  -h, --help           help for use
      --shell string   Shell to emit for: bash, zsh, pwsh (auto-detected if unset)
```
