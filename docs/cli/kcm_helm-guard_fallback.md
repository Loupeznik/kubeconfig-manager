## kcm helm-guard fallback

Toggle the pattern-less global fallback (compare path tokens directly when no pattern matches)

```
kcm helm-guard fallback <on|off> [flags]
```

### Options

```
      --dir string    Kubeconfig directory (default: ~/.kube)
      --file string   Apply only to this kubeconfig (default: global)
  -h, --help          help for fallback
```
