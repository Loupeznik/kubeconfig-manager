## kcm list

List kubeconfig files in the managed directory

```
kcm list [flags]
```

### Options

```
      --dir string   Kubeconfig directory (default: ~/.kube)
  -h, --help         help for list
  -v, --verbose      Show files that were skipped because they couldn't be parsed as kubeconfigs
```
