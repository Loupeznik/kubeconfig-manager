## kcm alert enable

Enable alerts for a kubeconfig (populates default blocked verbs)

```
kcm alert enable <file> [flags]
```

### Options

```
      --dir string   Kubeconfig directory (default: ~/.kube)
  -h, --help         help for enable
```
