## kcm uninstall-shell-hook

Remove the shell integration block from the rc file

```
kcm uninstall-shell-hook [flags]
```

### Options

```
  -h, --help           help for uninstall-shell-hook
      --rc string      rc file path (default depends on shell)
      --shell string   bash, zsh, or pwsh (auto-detected if unset)
```
