## kcm helm-guard add-pattern

Append one or more patterns to the existing list

```
kcm helm-guard add-pattern <pattern...> [flags]
```

### Options

```
      --dir string    Kubeconfig directory (default: ~/.kube)
      --file string   Apply only to this kubeconfig (default: global)
  -h, --help          help for add-pattern
```
