## kcm tag add

Add tags (file-level, or --context for one context only)

```
kcm tag add <file> <tag...> [flags]
```

### Options

```
      --allow-new        Add tags to the palette automatically if not present
      --context string   Apply to this context only (default: file-level)
      --dir string       Kubeconfig directory (default: ~/.kube)
  -h, --help             help for add
```
